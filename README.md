# TodoApi sample

Минимальное web-приложение на C#, предоставляющее CRUD API для управления задачами и использующее SQLite как встроенную базу данных.

## Запуск

```bash
cd TodoApi
DOTNET_ENVIRONMENT=Development dotnet run
```

Приложение слушает порт `http://localhost:5000` по умолчанию. После запуска база данных `todo.db` будет создана автоматически.

## Эндпоинты

| Метод | Путь | Описание |
| --- | --- | --- |
| `GET` | `/todos` | Список всех задач. |
| `GET` | `/todos/{id}` | Получение задачи по идентификатору. |
| `POST` | `/todos` | Создание новой задачи. Тело: `{ "title": "string", "notes": "string?", "isComplete": false }`. |
| `PUT` | `/todos/{id}` | Полное обновление существующей задачи. |
| `DELETE` | `/todos/{id}` | Удаление задачи. |

В режиме разработки по адресу `/swagger` доступна интерактивная документация.
